# Mi Primer Web

## Descripción

Este es un proyecto web personal que incluye una sección "Acerca de mí". Es un sitio sencillo diseñado para presentar información personal y mostrar habilidades en desarrollo web.

## Estructura del Proyecto

- **acerca-de-mi/**: Página principal del sitio
  - `index.html`: Estructura HTML de la página
  - `style.css`: Estilos y diseño
  - `fonts/`: Fuentes personalizadas
  - `images/`: Imágenes utilizadas en el proyecto

## Tecnologías Utilizadas

- HTML5
- CSS3

## Cómo Usar

1. Abre el archivo `acerca-de-mi/index.html` en tu navegador web
2. Navega por el contenido disponible

Proyecto desarrollado como parte del aprendizaje en desarrollo web.
